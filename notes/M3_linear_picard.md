# M3 — 线性算子的 `Ψ_I`：SVD → Picard，与四算子对照表

> prompt 02 / primer §3.2。线性腐蚀下，无权模数 `Ψ_I` 就是 **Picard 条件**；SFBD 的悲观、
> OMNI 的可识别性二分、以及「需要 clean 先验」统一成 `Ψ_I` 的三档。

## 推导（一页）：SVD → Picard

设线性腐蚀的算子矩阵为 `M`（= `operators.modulus_matrix`：确定性算子取 `A`，加噪算子取
`C_ν · A`）。模数定义

```
Ψ_I(φ) = inf{ ‖ψ‖₂ : Mᵀ ψ = φ }.
```

这是带线性约束的最小范数问题，解为伪逆 `ψ = (Mᵀ)⁺ φ`。代入并用 SVD `M = U Σ Vᵀ`：

```
Ψ_I(φ)² = ‖(Mᵀ)⁺ φ‖² = φᵀ (M Mᵀ)⁺ ... = φᵀ (Vᵀ)⁺Σ⁻²(V)⁺ φ = Σ_i ⟨φ, v_i⟩² / σ_i².
```

即 **Picard 条件**：恢复难度由奇异值 `σ_i → 0` 的速度对比目标在该方向的能量 `⟨φ,v_i⟩²`
决定。

**三档**（primer §3.2）：

| 谱情形 | `Ψ_I` | 物理含义 |
|---|---|---|
| `σ_i` 全部 O(1) | 有限温和 | 良态，少量样本即可 |
| `σ_i → 0`（但 >0） | 有限但巨大 | 病态（模糊 / 高斯高频），需大量样本 |
| `σ_i = 0` 且 `⟨φ,v_i⟩≠0` | `∞` | 非单射方向（灰度 / masking），**必须 clean 先验** |

特例自动掉出：
- **加噪**：`M=C_ν`，Fourier 乘子，`σ_i=Φ_ν(u_i)`，回收 M1（见 `M1_prop1_recovery.md`）。
- **模糊**：循环卷积，高频奇异值快衰减 → 有限但巨大。
- **灰度**：低秩投影，部分 `σ_i=0` → 落在 range 外的 `φ` 给 `∞`，方向集 ↔ OMNI 的歧义解集 `S(q)`。
- **masking**：对角 0/1，被 mask 的坐标 `σ_i=0`。

## 四算子对照表（dim=64，`psi_closed_form`）

测试函数：`DC`=常数模、`low-freq`=cos(k=2)、`high-freq`=cos(k=24)、`impulse`=δ₀。

| 算子 | DC | low-freq | high-freq | impulse | 代理 (RMS) |
|---|---|---|---|---|---|
| gaussian σ=0.4 | 1 | 1.003 | 1.559 | 1.382 | 1.382 |
| gaussian σ=1.0 | 1 | 1.019 | 16.05 | 32.72 | 32.72 |
| blur std=1.5 | 1 | 1.044 | 513.9 | 3.57e+04 | 3.57e+04 |
| grayscale rank=32 | ∞ | ∞ | ∞ | ∞ | 1 |
| masking α=0.4 | ∞ | ∞ | ∞ | 1 | 1 |

读法：
- **gaussian / blur**：`Ψ_I` 随频率与噪声/模糊强度增大——「有限但巨大」，正是 SFBD 的悲观来源。
- **grayscale**：一般光滑测试函数都落在随机秩-32 投影的 range 外 → `∞`；其 range 方向 `Ψ_I=1`
  （代理=1），即"看得见的方向平凡、看不见的方向需先验"。
- **masking**：cos 模在被 mask 坐标有质量 → `∞`；而 `impulse` 落在未 mask 坐标 → `Ψ_I=1`。
- **代理**（`psi_proxy`=非零谱上 RMS 放大）对 gaussian/blur 与 impulse 闭式一致（循环算子下
  二者恒等），对非单射算子只度量 range 部分（=1），零空间另由 clean 预算处理。
