# M4 — clean-space 弱度量速率定理（D6）

> prompt 07 / primer §3, 决策 B。把 OMNI 的腐蚀空间收敛率翻译到 clean 空间，得到一个
> **弱（光滑化）度量**下的速率。卡点不在"算"，在严谨性——已显式标注待 Yu 复核。
> **记号**：本节用 **q-加权 `Ψ_q`**（让 χ²/L² 不等式精确成立），与实验线（D1–D5）用的
> 无权 `Ψ_I` **不是同一量**——见 §5 与"待 Yu"。命名：SFBD = SFBD (ICML'25)，与 OMNI Prop 1 区分。

---

## 1. 弱度量与可恢复度模数

`q := T_r p_data`。腐蚀算子的伴随（对 ψ 永远线性）：`(T_r*ψ)(x)=∫r(y|x)ψ(y)dy=E[ψ(Y)|X=x]`。

**q-加权模数**：`Ψ_q(φ) := inf{ ‖ψ‖_{L²(q)} : T_r*ψ = φ }`。

**弱度量**（决策 B；只对能被 `T_r*` 以有界 `Ψ_q` 表示的光滑测试函数取 sup）：
```
d_R(p, p_data) := sup_{φ : Ψ_q(φ) ≤ R} | E_{p_data}[φ] − E_p[φ] |.
```

---

## 2. χ²/L² 伴随稳定性（决策 A）

对 `φ = T_r*ψ`，由伴随 + Cauchy–Schwarz：
```
|E_{p_data}[φ] − E_p[φ]| = |E_q[ψ] − E_{T_r p}[ψ]| ≤ ‖ψ‖_{L²(q)} · √(χ²(T_r p ‖ q)).
```
对满足 `T_r*ψ=φ` 的 ψ 取 inf，再对 `Ψ_q(φ)≤R` 取 sup：
```
d_R(p, p_data) ≤ R · √(χ²(T_r p ‖ q)).                                   (★)
```

---

## 3. 散度桥接引理（red line 4 — 不可略，但可证）

OMNI 的率控制 `KL(q ‖ T_r p^k)`（q 在前，KL），而 (★) 需要 `χ²(T_r p^k ‖ q)`（q 在后，χ²）。
**方向 + 类型都不同，无无条件单边界。** 在**密度比有界**假设下可桥接：

> **假设 (BDR)**：存在 `0<c≤C<∞` 使 `c ≤ (T_r p^k)/q ≤ C`（即 `T_r p^k` 与 `q` 等价且比值有界）。

> **引理（桥接）**：设 `m, q` 概率密度，`t:=m/q ∈ [c,C]`。令
> `β(c,C) := min_{t∈[c,C]} (−log t + t − 1)/(t−1)² > 0`。则
> ```
> χ²(m ‖ q) ≤ (1/β(c,C)) · KL(q ‖ m).
> ```

**证明**：令 `g(t) = −log t + t − 1`，凸、`g(1)=0`、`g''=1/t²>0`，故 `g(t) ≥ β(c,C)(t−1)²`
于 `[c,C]`。因 `m` 是概率密度，`E_q[t]=∫m=1`，于是
`E_q[g(t)] = E_q[−log t] + E_q[t−1] = KL(q‖m) + 0`。
又 `E_q[g(t)] ≥ β E_q[(t−1)²] = β·χ²(m‖q)`。两式合并即得。∎

**数值核验**（脚本见 notes 末）：
- `β(0.5,2)=0.307`（桥接常数 `1/β=3.26`）、`β(0.2,5)=0.149`（6.69）、`β(0.1,10)=0.083`（12.1）。
  → `C` 越大 `β→0`，桥接常数发散，**故 BDR 必须有界**（决策 A 与 OMNI KL 率之间的残留张力）。
- 随机抽 2000 对有界比高斯，`χ²(m‖q) ≤ KL(q‖m)/β` **全部成立**（最紧比 0.978 ≤ 1）。

## 4. 主定理（D6）

> **定理（clean-space 弱度量率）**：设 OMNI/Online-SFBD-OMNI 迭代 `{p^k}` 满足 Eq 21
> `min_{1≤k≤K} KL(q‖T_r p^k) ≤ KL(h†‖p^0)/(γK)`（γ 为 OMNI 步长参数），`k*` 为达最小的迭代。
> 若 `T_r p^{k*}` 满足 (BDR)（常数 `c,C`，桥接常数 `β=β(c,C)`），则
> ```
>   d_R(p_data, p^{k*}) ≤ R · √( KL(h†‖p^0) / (β γ K) ).
> ```

**证明骨架**：(★) → 代入桥接引理 `χ²(T_r p^{k*}‖q) ≤ KL(q‖T_r p^{k*})/β` → 代入 Eq 21。∎

**读法**：弱度量误差 `O(1/√K)`；常数含 (i) **可恢复度档位 `R`**（越病态越大）、(ii) **桥接 `1/√β`**
（密度比越宽越大）。这是两篇都没有的"腐蚀空间率 → clean 空间率"翻译，且把代价显式写成
模数 × 桥接。

## 5. W2 / TV 推论（决策 B；偏差项不可省，red line 2）

`d_R` 只控制 `Ψ_q(φ)≤R` 的光滑测试函数。W2（需 1-Lipschitz 测试族）/ TV（需指示函数）的 `φ`
在病态 `T_r` 下一般 `Ψ_q(φ)=∞`（高频内容）。**截断**处理：把目标 `φ` 用带限 `φ_U`（频率 `|u|<U`）逼近：
```
|E_{p_data}φ − E_p φ| ≤ |E[φ−φ_U]|  +  d_{R(U)}(p_data, p)
                     ≤  bias(U)      +  R(U)·√(KL(h†‖p^0)/(βγK)).
```
- **bias(U)**：逼近偏差，由**目标密度光滑性**控制（φ 的高频尾 / 密度的 Sobolev 范数）；`U↑` 时 ↓。
- **R(U) = sup_{|u|<U} Ψ_q(e_u)**：带限测试族的模数上界，`U↑` 时 ↑。对加性高斯
  `R(U) ~ exp(σ²U²/2)`（M1）；对 ordinary-smooth 多项式增长。
- 对 `U` 优化 `bias(U) + R(U)/√(βγK)`：
  - **supersmooth（高斯）**：`R(U)` 指数 → 最优 `U ~ √(log K)` → **`(log K)` 型慢率**——回收
    SFBD 的悲观速率（红线 1：只回收、不重证）。
  - **ordinary-smooth**：`R(U)` 多项式 → **多项式率**。
- **诚实**：W2/TV 控制**必须**带 `bias(U)` 项与目标光滑性假设；裸 `d_R` 不等于 W2/TV。

## 6. 与实验线（D2/D3）的关系（交叉验证）

- M4 是**算法迭代** `K`（腐蚀空间 → clean 弱度量）的率；D2/D3 是**样本复杂度** `n,m`（估计层）的律。
  二者**互补**，共享同一个标度量——可恢复度模数 `R`（=Ψ）。
- 结构一致性：M4 误差 `∝ R·(腐蚀空间率)^{1/2}`，D3 验证了 `n ∝ Ψ²/ε²`（`results/D3_*`）。两处
  "误差 ∝ 模数 × (预算)^{-1/2}" 同型。
- 桥接引理已**独立数值核验**（§3）。直接数值验证主定理需在玩具上跑 OMNI 交替算法（当前范围外，
  可作小增量）。

## 7. 待 Yu 复核（核心请教点）

1. **泛函分析**：`range(T_r*)` 是否闭、`Ψ_q` 的 inf 是否可达（取到 min）——决定 `Ψ_q` 良定。
2. **BDR 能否弱化**：密度比有界是否可换成局部/矩条件？桥接常数 `β(c,C)` 是否有更优形式？这是
   决策 A（χ²/L²，为对接 SVD/Picard）与 OMNI 的 `KL(q‖·)` 率之间的残留张力。
3. **`Ψ_q ↔ Ψ_I`**：实验用无权 `Ψ_I`、定理用 q-加权 `Ψ_q`，二者关系（广义 SVD？常数比？）——
   否则定理与 D2/D3 数值无法直接对齐。
4. **§5 截断优化**：`bias(U)+R(U)/√(·)` 的 `U`-优化是否有他偏好的标准技术（Sobolev / 带限）。
