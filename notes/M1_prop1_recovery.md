# M1 — `Ψ_I(e_u) = exp(σ²‖u‖²/2)`：退回 SFBD (ICML'25) Prop 1

> 第 1 周自检（prompt 02 / primer §3）。结论：本框架的无权模数 `Ψ_I` 在「高斯加噪 +
> 特征函数测试族」上**精确**回收 SFBD 的去卷积放大因子。这是框架正确的最强信号。

## 推导（半页）

加性高斯噪声 `y = x + ε`，`ε ~ ν = N(0, σ²)`。腐蚀算子是卷积 `T_r p = p * ν`，其
**伴随**作用在测试函数上仍是卷积（ν 对称）：

```
(T_r* ψ)(x) = ∫ ν(y−x) ψ(y) dy = (ψ * ν)(x).
```

Fourier 域 `Φ_{T_r*ψ}(u) = Φ_ψ(u) · Φ_ν(u)`，其中 `Φ_ν(u) = exp(−σ²u²/2)`。

要用 `T_r*ψ` 表示纯频率测试函数 `φ = e_u`，需 `Φ_ψ(u) = 1/Φ_ν(u)`，即最小范数预像

```
ψ = e_u · exp(σ²u²/2)      ⟹     Ψ_I(e_u) = inf‖ψ‖ = 1/Φ_ν(u) = exp(σ²u²/2).
```

更一般地，加性噪声下 `T_r*` 是 Fourier 乘子，故 `Ψ_I(e_u) = 1/Φ_ν(u)`：高斯
(supersmooth) → `exp` 爆炸；Laplace (ordinary-smooth) → 多项式衰减。**经典反卷积二分法
自动回收**（只引用 Fan'91 / Meister'09，不重证——primer 红线 1）。

## 数值确认（`experiments/e1_verify_psi.py`）

把高斯加噪算子表示为 Fourier 乘子循环矩阵 `C_ν`（`operators.gaussian_conv_matrix`，
特征值精确为 `exp(−σ²u_k²/2)`，无混叠），对每个频率模 `e_{u_k}` 用 Picard 闭式
`Ψ_I(φ)²=Σ⟨φ,v_i⟩²/σ_i²` 数值算 `Ψ_I`，与解析 `exp(σ²u_k²/2)` 对照。

结果（dim=128, σ=0.5）：

| 量 | 值 |
|---|---|
| `max rel_err`（数值 vs 解析） | **9.8e-16**（机器精度） |
| 拟合斜率（`log Ψ_I` vs `u²`） | **0.125000** |
| 目标 `σ²/2` | **0.125000** |
| 斜率相对误差 | **4.4e-16** |

图：`results/D1_M1_prop1_recovery.png`（左：`log Ψ_I` vs `u²` 落在斜率 `σ²/2` 直线上；
右：相对误差在机器精度量级）。
